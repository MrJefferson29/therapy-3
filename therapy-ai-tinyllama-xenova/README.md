# Therapy AI TinyLlama Model

This is a fine-tuned TinyLlama model for therapy applications, optimized for use with @xenova/transformers.

## Model Information
- Base Model: TinyLlama-1.1B-Chat-v1.0
- Fine-tuned for: Therapy and mental health conversations
- Format: Compatible with @xenova/transformers
- Size: ~1.1B parameters

## Usage
This model is designed to be used with the @xenova/transformers library in JavaScript/Node.js environments.

## Files
- config.json: Model configuration
- model.json: Additional model metadata
- tokenizer.json: Tokenizer configuration
- tokenizer_config.json: Tokenizer settings
- special_tokens_map.json: Special tokens mapping
- generation_config.json: Text generation settings

## Compatibility
- @xenova/transformers
- JavaScript/Node.js
- Web browsers
- Server-side applications
